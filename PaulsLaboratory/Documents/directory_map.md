## File Type Legend
- [.swift] = Swift source code files
- [.md] = Markdown documentation files
- [.storyboard] = iOS Interface Builder storyboard files
- [.plist] = Property list configuration files
- [.json] = JSON configuration files
- [.pbxproj] = Xcode project configuration
- [.xcworkspacedata] = Xcode workspace configuration
- [.xcplayground] = Swift playground configuration
- [executable] = Compiled executable binary
- [git-repo] = Git repository folder
- [user-data] = Xcode user-specific data (excluded from version control)
